# Minimal SEO Theme Premium

Tema WordPress ultraligero para nichos y SEO de marca (modelo Órbita).

**Versión:** 2.6.8  
**Repositorio:** [github.com/saritacanosierra/minimal-seo-theme](https://github.com/saritacanosierra/minimal-seo-theme)

## Instalación rápida

1. Subir `minimal-seo-theme.zip` en **Apariencia → Temas → Añadir nuevo**
2. Activar el tema
3. Leer [doc/GUIA-MAPA-DECISIONES-CLIENTE.md](doc/GUIA-MAPA-DECISIONES-CLIENTE.md) o abrir **Apariencia → Mapa de decisiones**
